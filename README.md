<h1>Programmer-Art version of a salad chef simulation in Unity.</h1>

<h3>Unity Version 2019.2.11</h3>

<p>Control a Player 1 and Player 2 Using Same Keyboard</p>
<ul><li>A for left move Player1</li>
    <li>D for right move Player 1</li>
    <li>W for go up Player 1</li>
    <li>S for go down Player 1</li>

  </ul>

<ul><li>J for left move Player2</li>
    <li>L for Right move Player2</li>
    <li>K for go down move Player 2</li>
    <li>I for go UP move Player2</li>

</ul>
